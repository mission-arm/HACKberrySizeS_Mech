# HACKberrySizeS_Mech

小児向けサイズの HACKberry の、機構部品の3Dプリント用STLファイルです。

ディレクトリ構成は以下のようになっています。

HbSizeSMech/
│  HbSizeS_BOM.xlsx : 小児用 HACKberryの部品一覧エクセルファイルです。
│  README.md        : この説明書です。
│
├─UnitHand/ : 手部分の部品のフォルダです。
│  ├─JointForWrist/ : 手首との接続部分のフォルダです。
│  │      HbWristMountForChild_aiwa.stl

│  │
│  └─Unit_SR-H/ : 手のメインの部品類のフォルダです。
│      ├─Parts_SR-H/ : 手のひら, 手の甲の部品のフォルダです。

│      │
│      ├─Unit_SR-I/ : 人差し指の部品のフォルダです。
│      │      SR-I-01.stl
